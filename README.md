# 📝 To-Do List on Blockchain (Soroban Smart Contract)

## 📌 Project Description

This project is a decentralized To-Do List built using Soroban smart contracts on the Stellar blockchain. It allows users to manage tasks in a secure, transparent, and immutable environment.

Unlike traditional to-do apps, this solution stores tasks on-chain, ensuring that data cannot be tampered with and remains globally accessible.

---

## ⚙️ What It Does

The smart contract allows users to:

- Add new tasks to the blockchain
- View all stored tasks
- Remove specific tasks
- Clear all tasks

All operations are executed on-chain using Soroban, ensuring trustless interaction without relying on centralized servers.

---

## ✨ Features

- ✅ Decentralized task management  
- 🔒 Immutable and tamper-proof storage  
- ⚡ Built on Soroban (fast and efficient smart contracts)  
- 📦 Simple and minimal design for learning and extension  
- 🧩 Easy integration with frontend apps (React, etc.)  
- 🗑️ Task deletion and clearing support  

---

## 🚀 Smart Contract Functions

| Function        | Description                     |
|----------------|---------------------------------|
| `add_task`     | Adds a new task                 |
| `get_tasks`    | Retrieves all tasks             |
| `remove_task`  | Removes a task by index         |
| `clear_tasks`  | Deletes all tasks               |
